## Summary

This lab successfully demonstrated the detection of an SSH brute-force attack using Wazuh SIEM.

---

## Key Takeaways

- Authentication logs provide valuable evidence during investigations.
- Wazuh can detect repeated authentication failures in near real time.
- Centralized log collection improves visibility across monitored systems.
- Repeated failed login attempts should always be investigated.
